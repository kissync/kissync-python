A cross platfom file synchronization tool for the SmartFile cloud
=======

A cross platfom file synchronization tool for SmartFile


Running Kissync
===================


    $ git clone https://github.com/travcunn/kissync-python.git kissync
    $ cd kissync
    $ python main.py
